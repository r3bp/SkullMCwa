What is SkullMC?

SkullMC is a Minecraft server that supports both Java Edition and Bedrock Edition, making it accessible to players on all platforms. It offers a fun and exciting experience with unique game modes and a welcoming community for players of all skill levels.  

User friendly

SkullMC provides a smooth and enjoyable experience for all players, with an easy-to-use interface and a supportive community to help you adapt quickly. 

Up to date

The server is always up-to-date, ensuring the latest features and an exceptional gaming experience. Join us now at play.skullmc.xyz for something truly unique.

Fast support

Our support team is always ready to assist you and resolve any issues as quickly as possible, ensuring you enjoy uninterrupted gameplay. 

How to join

Here you can see, how you can join on the Minecraft server.

1

Add the server

Go to Multiplayer in Minecraft, then Add Server
2

Enter server adress

Enter play.skullmc.xyz in the server address field and click "Done".
3

You're done!

Now you can play on SkullMC with other players!
Story about SkullMC

SkullMC started as a small idea to create a fun and welcoming community for all Minecraft players. Supporting both Java Edition and Bedrock Edition, the server is designed to provide the best possible experience for players on any platform. 

With unique game modes like PvP, Bedwars, and Duels, SkullMC constantly evolves to bring exciting updates and features. It’s more than just a server—it’s a place where players connect, compete, and grow. Join us and become part of the SkullMC story!
